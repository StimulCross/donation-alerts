    Donation Alerts - Common

    This package provides a set of shared utilities and type definitions used across the @donation-alerts packages. Although it is primarily intended for internal use, you might find some of its types and utilities useful in your own code.

    Installation

    Using npm:

    npm i @donation-alerts/common

    Using yarn:

    yarn add @donation-alerts/common

    Using pnpm:

    pnpm add @donation-alerts/common

    For more detailed information, please check the documentation.
